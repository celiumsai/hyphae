# Public clients for API v1

Every client below consumes only the public `/v1` contract. None opens a data
directory, imports storage/query internals, or requires a database, model, or
embedding provider.

## Rust

```rust,no_run
use hyphae_client::HyphaeClient;
use hyphae_contracts::v1::GetRequestV1;

# async fn example() -> Result<(), Box<dyn std::error::Error>> {
let client = HyphaeClient::builder("http://127.0.0.1:8787")?.build()?;
let response = client.get(&GetRequestV1 { key_hex: "61".into() }).await?;
println!("found={} request={}", response.value.found, response.request_id);
# Ok(())
# }
```

## TypeScript

```typescript
import { HyphaeClient } from "@celiums/hyphae";

const client = new HyphaeClient("http://127.0.0.1:8787");
const response = await client.get({ key_hex: "61" });
console.log(response.value.found, response.requestId);
```

Hyphae transports signed 64-bit JSON integers. Values within JavaScript's safe
range are `number`; larger values are `bigint`. Sending an unsafe `number`
throws locally, so use `9223372036854775807n` when exact large values matter.

```bash
cd sdks/typescript
npm ci --ignore-scripts
npm test
```

## Python

```python
from hyphae_sdk import HyphaeClient

client = HyphaeClient("http://127.0.0.1:8787")
response = client.get({"key_hex": "61"})
print(response.value["found"], response.request_id)
```

The Python package requires Python 3.11 or newer and has no runtime
dependencies. Python integers preserve the complete signed 64-bit domain.

## CLI

Request bodies are typed contract JSON read from a file or standard input.
Bearer secrets come from `HYPHAE_BEARER_TOKEN` or a restricted file, never an
argv value.

```bash
hyphae remote --base-url http://127.0.0.1:8787 capabilities
printf '%s' '{"key_hex":"61"}' |
  hyphae remote --base-url http://127.0.0.1:8787 get --request -
```

`put`, `get`, `delete`, and `query` print the exact v1 success model. `witness`
writes to a new path and refuses to replace an existing file.

## MCP

Configure an MCP host to spawn:

```json
{
  "command": "hyphae",
  "args": ["mcp", "--base-url", "http://127.0.0.1:8787"]
}
```

The stdio adapter implements stable MCP `2025-11-25` and exposes
`hyphae_capabilities`, `hyphae_put`, `hyphae_get`, `hyphae_delete`, and
`hyphae_query`. Mutations remain subject to host/user authorization. The
adapter does not start a listener or access storage internals.

## Conformance

After building the Rust binaries and TypeScript SDK:

```bash
cargo build -p hyphae-cli -p hyphae-conformance-rust --locked
(cd sdks/typescript && npm ci --ignore-scripts && npm run build)
python tools/run_conformance.py
```

Each consumer receives a fresh server and data directory. Success emits one
JSON summary containing all five clients.
